PROBLEMAS?
================================================================================

Se nao funcionar:

1. Verifique se Node.js esta instalado:
   - Abra CMD
   - Digite: node --version
   - Se nao funcionar, baixe em: https://nodejs.org

2. Verifique se .env esta correto:
   - Abra .env com Bloco de Notas
   - Verifique todas as credenciais
   - Salve

3. Leia TROUBLESHOOTING.md para mais ajuda
